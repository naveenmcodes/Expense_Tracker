Project Name: Expense Tracker

Scope: A Python-based Expense Tracker that allows users to record, manage, and analyze daily expenses. The application stores expense data in CSV format and provides spending summaries through a simple menu-driven interface.

Features:
- Add new expenses
- Automatic date recording
- Expense categorization (Food, Travel, Study, Entertainment, etc.)
- View all expenses
- Calculate total spending
- Category-wise spending summary
- Delete expense records
- CSV-based data storage

Technologies Used:
- Python
- CSV Module
- datetime Module

Data Storage:

Expense data is stored in a CSV file named:expenses.csv
